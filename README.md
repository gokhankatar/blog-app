## Installation

To clone the repository, run the following command:

```sh
git clone https://github.com/gokhankatar/blog-app

``` 
go to directory

```sh
cd .\blog-app\

``` 
and then install npm:

```sh
npm install -f

``` 
open your browser:

```sh
npm run dev
